Exibe cupons de desconto personalizados para usuários logados no WooCommerce.

== Description ==
O plugin oferece uma solução flexível para lojas online que desejam proporcionar uma experiência de compra mais personalizada. Ao usar este plugin, os proprietários de lojas podem:
– **Exibir Cupons para Usuários Logados**: Os usuários que estiverem logados poderão ver cupons personalizados que são aplicáveis a eles, com base no e-mail associado à sua conta.
– **Adicionar Códigos de Cupons Específicos**: Permite que os administradores adicionem códigos de cupons específicos que podem ser visualizados por todos os clientes, facilitando campanhas promocionais.
– **Remover Cupons Expirados**: O plugin automaticamente exclui cupons que já foram utilizados ou que estão fora do período de validade, assegurando que os usuários sempre vejam cupons relevantes.
– **Integração com WooCommerce**: Totalmente integrado ao WooCommerce, este plugin é fácil de configurar e gerenciar, permitindo que você foque em vender e atender seus clientes.


Exibe cupons personalizados e válidos para usuários logados no WooCommerce, com fácil integração e gestão.

Recursos principais:
– Exibe cupons disponíveis para o usuário logado com base no seu e-mail.
– Permite adicionar códigos de cupons específicos para exibição.
– Exclui automaticamente cupons expirados ou já utilizados.
– Integração fácil com o WooCommerce.
– Configuração simples via painel do WordPress.

Use o shortcode `[cpus_show_my_coupons]` em qualquer página ou postagem para exibir os cupons disponíveis.

== Instalação ==

1. Faça o upload dos arquivos do plugin para o diretório `/wp-content/plugins/cupons-personalizados-para-usuarios` ou instale diretamente via a tela de plugins do WordPress.
2. Ative o plugin através da tela "Plugins" no WordPress.
3. Acesse **Configurações > Cupons Personalizados** para adicionar códigos de cupons específicos.
4. Use o shortcode `[cpus_show_my_coupons]` em qualquer página para exibir os cupons.

== FAQ ==

= Como adiciono códigos de cupons específicos para exibição? =
Após ativar o plugin, vá até **Configurações > Cupons Personalizados** e adicione os códigos de cupons desejados, separados por vírgula.

= Posso exibir os cupons em qualquer página? =
Sim, basta inserir o shortcode `[cpus_show_my_coupons]` na página ou postagem onde deseja exibir os cupons.

= O plugin é compatível com qualquer tema WooCommerce? =
Sim, o plugin foi desenvolvido para funcionar com qualquer tema WooCommerce que siga os padrões de desenvolvimento do WordPress.
